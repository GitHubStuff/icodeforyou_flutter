# Changelog

## [0.0.2] - 2025-11-08

### Added
- Initial release of splash_screen_package
- AppSplashScreen widget with customizable fade-in animation
- SplashScreenCubit for state management using flutter_bloc
- LoadingError class for error handling
- Timeout handling with configurable duration
- Custom error widget builder pattern for maximum flexibility
- Comprehensive test coverage (100% LCOV)
- Full documentation with usage examples

### Features
- Black screen splash with customizable fade-in animation and curve
- Loading spinner with configurable barrier overlay
- Error handling with slide-in transition
- Cross-fade transition to main app on successful initialization
- Back button blocking during splash screen display
- SOLID principles and Clean Code architecture
- Fully testable design with 100% code coverage

### Dependencies
- flutter_bloc: ^9.1.1
- equatable: ^2.0.5
